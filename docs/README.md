# Docs

The docs folder is, of course, for documentation related to the project. At first it will be sparse, but over time, I will try to make it more useful.

## ficsiteconf.json

This file is an example config file for the project. In fact, during development I will point to it on my local machine. Keys and such that I commit in this file will not be used on a production server. They are intended as examples so that you can have a template to fill out for production without having to guess.

At first, some of the stuff will just be nonsense, as I figure out exactly what is acutally needed for the project, and as I build up the project.
